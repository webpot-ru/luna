# Deck Specs Registry

Этот каталог является registry для спецификаций колод перед генерацией.

Source of truth для operational order, `Sort` и статусов остается [Deck Master Plan](../deck-master-plan.md). Этот registry хранит только физические deck specs: scope, include/exclude, target range, level, frequency, priority, quality profile, candidate-pool rule, next deck and QA notes.

Правило:

- `candidate` и `planned` могут существовать в Deck Master Plan без spec-файла;
- `spec_ready`, `approved_for_generation` и `generated` обязаны иметь строку в registry и spec-файл;
- `planned` не означает готовность к генерации;
- spec-файл обязателен до генерации новой колоды;
- spec-файл обязан иметь `deck_profile` and `risk_flags`; они управляют batch size, pre-import blockers and required QA families;
- `approved_for_generation` требует candidate-pool evidence в `outputs/candidate-pools/<set_id>_candidate_pool.jsonl`;
- для `approved_for_generation` каждая selected candidate-pool row должна фиксировать `scope_decision`, `duplicate_reuse_decision`, `compound_multiword_risk` and `required_qa_profile`;
- before approval/generation, `scripts/check-word-selection-quality.mjs <Sort|set_id>` must pass. It blocks weak selected rows such as ambiguous words without disambiguation, compounds without whole-meaning risk, generic examples, unresolved duplicate decisions and selected rows that do not fit the deck scope;
- generation-ready check обязателен перед созданием новых карточек и должен проходить только для `approved_for_generation`;
- не создавать spec-файлы пачкой на весь backlog, пока колода не готовится к генерации.

Allowed `deck_profile` values:

```text
object_noun, action_verb, adjective_state, closed_set, number_quantity,
time_calendar, food_countability, document_admin, health_safety,
service_problem, regional_variant_heavy, transcription_high_risk
```

Проверка:

```bash
node scripts/check-deck-specs.mjs
node scripts/check-deck-ready.mjs <Sort|set_id>
node scripts/check-deck-candidate-pool.mjs <Sort|set_id>
node scripts/check-word-selection-quality.mjs <Sort|set_id>
```
